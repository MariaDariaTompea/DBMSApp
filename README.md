# DBMgmt — DBMS App

An interactive, premium-designed web application designed as a study companion for the **Database Management Systems (DBMS)** course at **Babeș-Bolyai University**.

---

## 🌸 What We Are Doing
We are building a highly visual, immersive interactive book that recaps core database principles:
1. **Interactive Manual**: Structure divided into:
   - **Course Recap Explained** (Data Types, Table Creation, Primary vs Foreign Keys).
   - **Seminars & Laboratories** (Step-by-step solved labs).
   - **Exam Papers & Solutions** (Practical raw coding exercises).
2. **Timeline View**: A draggable line with interactive lesson chapters.
3. **Lesson Book Viewer**: A virtual book displaying lesson pages, styled tables, code blocks, and diagrams.

## ✨ Premium Features
- **3D Sakura Petals & Tech Particles**: Fluid background animation rendering cherry blossoms and SQL commands.
- **3D Table Flip Translations**: Hovering over any table for 3 seconds triggers a 3D flip animation, translating the contents into **Japanese** to assist multilingual learners.
- **Analyst Tooltips**: Interactive context tooltips (e.g., data types advice for data analysts) that display cleanly without layout clipping.
- **Responsive & Modern Design**: Dark theme featuring glassmorphism elements, Outfit/Inter typography, and smooth micro-animations.

## 🛠️ Technology Stack
- **Structure**: Semantic HTML5
- **Style**: Custom Vanilla CSS (with custom keyframe animations and theme variables)
- **Logic**: Vanilla ES6+ Javascript
